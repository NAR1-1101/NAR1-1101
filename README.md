# Pulse Forge (Incremental Clicker)

ブラウザで動作するエネルギーコア育成インクリメンタルゲームです。クリック・自動生成・転生・イベントで成長を加速します。

## 起動方法
`index.html` をブラウザで開いてプレイできます。

## 実装済み機能
- 手動クリックで Energy 獲得
- 6段階設備の購入と指数コスト増加
- 自動生成 (EPS)
- 通常アップグレード
- 転生 + Shard
- 恒久アップグレード
- 実績
- ランダムイベント
- 統計表示
- LocalStorage セーブ/ロード
- オフライン進行報酬 (12時間上限)

## ファイル構成
- `index.html`: UI構成
- `styles.css`: レイアウト/演出
- `script.js`: ゲームロジック/保存/描画
